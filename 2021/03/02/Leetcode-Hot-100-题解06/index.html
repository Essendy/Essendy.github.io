题目： 正则表达式匹配（困难）

给你一个字符串 s 和一个字符规律 p，请你来实现一个支持 ‘.’ 和 ‘*‘ 的正则表达式匹配。

‘.’ 匹配任意单个字符
‘*’ 匹配零个或多个前面的那一个元素
所谓匹配，是要涵盖 整个 字符串 s的，而不是部分字符串。

注：

s 可能为空，且只包含从 a-z 的小写字母。
p 可能为空，且只包含从 a-z 的小写字母，以及字符 . 和 *。
保证每次出现字符 * 时，前面都匹配到有效的字符

示例1

输入： s = “aa” p = “a”

输出： false

解释 “a” 无法匹配 “aa” 整个字符串。

示例2

输入： s = “ab” p = “.*“

输出： true

解释 “.*“ 表示可匹配零个或多个（’*‘）任意字符（’.’）。

 

解题思路

首先我们定义dp[i][j]为s前i个字符与p前j个字符是否匹配, 此时考虑s[:i], p[:j],因此最后一个字符的索引为i-1, j-1。那么显然我们可以初始化dp[0][0]=True，因为两个空字符串必然是匹配的。接下来继续考虑，当s为空字符串时，只有p的偶数位全为’*‘时，s与p才能匹配。因此，可以确定当i=0时，dp[0][j] = dp[0][j-2] 如果 p[j-1] = ‘*‘。

由于s[i-1]必为小写字母 p[j-1]可能为字母，可能为‘.’, 也可能为’*‘.因此我们对p[j-1]分情况讨论

  1. 当p[j-1] ！= ‘/*’

    1. 如果p[j-1] = ‘.’, 它与s[i-1]必然匹配，所以dp[i][j] = dp[i-1][j-1]。
    2. 如果p[j-1] != ‘.’, 只有当s[i-1] = p[j-1]情况下我们才要去看s的前i-1个与p的j-1个是否匹配，即dp[i][j] = dp[i-1][j-1], 否则就是不匹配。

  2. p[j-1] 为’*’
    存在两种情况：

    1. 如果它与前一个字符的作用是表示前面字符有0个，那么dp[i][j] = dp[i][j-2]

    2. 如果它与前面字符表示有一个或多个，那么这个作用成立的必要条件是s[i-1]与p[j-2]匹配, 否则他们的作用只能表示0个。那么在表示一个或多个作用成立条件下，我们可以写出dp[i][j] = dp[i-1][j-2]

      那么换个角度，我们也可以将上面的两种情况根据s[i-1]与p[j-2]是否匹配这个条件讨论得到：

    3. 如果匹配， 它的作用可以是表示0个，或者表示一个或多个，即dp[i][j] = dp[i][j-2] or dp[i-1][j-2]

    4. 如果不=匹配，它的作用只能是表示0个，即dp[i][j] = dp[i][j-2]

总结如下

1. p[j-1] != ‘*’
    dp[i][j] = dp[i-1][j-1] if (p[j-1] == '.' or s[i-1] == p[j-1]) else False
2. p[j-1] == '*':
    dp[i][j] = dp[i-1][j] or dp[i][j-2] if p[j-1] == '.' or s[i-1]==p[j-2] else dp[i][j-2]
