题目： 盛最多水的容器（中等）

给你 n 个非负整数 a1，a2，…，an，每个数代表坐标中的一个点 (i, ai) 。在坐标内画 n 条垂直线，垂直线 i 的两个端点分别为 (i, ai) 和 (i, 0) 。找出其中的两条线，使得它们与 x 轴共同构成的容器可以容纳最多的水。

说明：你不能倾斜容器。

示例1

 

解题思路

经典面试双指针题，思路是每次向内移动对应较小值的指针，直到双指针重合。

证明如下：假设目前指针指向x,y 且 x < y, 那么此时它就是所有以x为一个边界的容器中的最大值$V^* = x \times d$, 这里d为两指针间隔距离。因为不管怎么把y对应的指针往内移动，容积始终有 $V = min(x, c)\times d’ < x \times d = V^*$。
